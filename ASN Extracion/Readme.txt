FatBoy v1.3 is small script helps netowrk personel & normal users to extract specific ASN(Autonomous System Number) prefixes as they advertised , convert them to thier original supernet addresses then adding them to router route table ( Mikrotik routers only ) in order to control traffic direction when there is more than one wan connection exist , created this script to help me in one of my projects , modifications & updates will follow.

Usage :
just run the .py file with python 3 , install the mentioned libraries & fill the requested information when they requested from the script ..Enjoy.
